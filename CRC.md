# Class, Responsiblity, Collaborators

## Card
Reponsibilities: 
- Knows Color
- Knows Number
- Knows if Special Card, and if so, which type

## Player
Reponsibilities: 
- Knows Score
- Knows which Deck it owns
- Knows if it is playing

## Deck
Reponsibilities: 
- Knows which cards it owns
- Knows the order of the cards
